概貌

这个应用程序的目的是模型之间的轨道关系地球和太阳导致了我们所看到的相对太阳运动。以及在选定的站点位置显示完整 的3D太阳路径图，您可以轻松切换以地理为中心和以天真为中心的观点之间，以及覆盖一些常用的信息了解关系的各种特征。

例如，打开“Twilight”和“Circles”覆盖图，然后选择 '常用的日期'菜单中的'Summer Soltice'(). 点击 'Play' 按钮 () 时间的动画，然后看看北方 和南极明确了解为什么北极和南极圈位于它们所在的位置。 你可以做一些类似于'Sub-Solar'和'Tropics'的选项。有几个 更值得为自己探索的东西，比如你调整太阳路径的时候 现场纬度和看到赤纬角如何随日期变化。值得，如果你可以花一个 有时间玩耍并试验模型。

背景

这是Processing中的一个Java applet的另一个HTML5版本。 最近在我的太阳能计算中做了一些低级别的优化，我需要更好的 以实际的方式看到代码的实际行为，并通过交互式的方式来实现它的步伐。甚至 最广泛的测试套件永远不会替代良好的动手可视化。

只是做这个应用程序，我发现真正有用，甚至有见地。加工 用不同的方式来展示各种特征以及如何最好地处理这两个行星 预测实际上改变了我对计算的想法，并导致了一些有用的结果 改进。希望其他一些人可能会发现它同样有用作为一种更好的方式 了解太阳运动。

有趣的功能

以下是这个应用程序的一些更有趣的功能 我实现了相当多的乐趣：

  • 地心视图中的模拟气氛，其发光跟踪太阳位置并消失 当在阴影中时,
  • 当星场和太阳位置接近无穷远时 平移和移动视图而不影响近/远深度缓冲区太大,
  • 纹理融合了地球上日光和阴影区域，显示夜间照明 它旋转,
  • 当太阳能地平线选项覆盖时，增强的晨昏带
  • 由于太阳能的大小，可视化几何和实际日出/日落之间的区别 磁盘和大气折射。

查看操作

您可以使用 鼠标 笔 或 手写笔, 或通过 触摸 平板电脑或手机. 交互式方式调整模型的3D视图, 您也可以使用 视图设置弹出框 中的项目.

旋转:
使用 左/右按钮 或 单个手指拖动,
或使用 箭头 键.
平移:
使用鼠标中键 或 两个/三个手指拖动,
或使用 A 和 D 键.
缩放:
使用 滚轮 或用 两个手指捏住,
或使用 W 和 S 键.

注: 你可以使用 Shift 和 Ctrl/Meta键来调整每个滚动事件或按键的增量.

键盘修饰符

Shift 和 Ctrl/Meta 键被广泛用于修改交互式数据输入。这适用于所有增量按钮，滚轮运动， 滑块控件和输入元素.

Shift 键:
将输入增加到较大值,通常为十倍或明显较大的增量,例如日期为1个月,时间为1小时.
Ctrl 或 Meta 键:
将输入值减小到较小,通常为十分之一或最小合理增量 如日期为1天或时间为1分钟.

注: 您可以使用 滚轮 来编辑数据值，以便将鼠标悬停在任何滑块，数字输入或表格行上以指示其可编辑性.
